Python-Pandas para Psicólogos y No-Programadores.
Este curso está diseñado para que aprendas Python a tu favor y puedas analizar cualquier base de datos que tengas. Si estás cansado de depender de macros de Excel o sufrir incompatibilidades, aquí verás cómo usar Python desde cero para crear y manipular tus datos. Al final tendrás esa sensación de poder —como automatizar hasta el proceso de pan tostado (✌)— y ordenar tus datos para analizarlos cuando quieras.
